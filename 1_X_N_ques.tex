\item prove algebraically the type of symmetry 
\end{itemize}

\begin{question}[class=easy]
Let $f(x)=x^{4}-2x^{2}+1$.
\begin{enumerate}
\item Have Desmos graph $f(x)$. Copy a sketch.
\item What type of symmetry does $f(x)$ appear to have?
\item Use Desmos to test for symmetries by graphing $f(-x)$ and $-f(-x)$. Which test produces a graph identical to the original $f(x)$?
\item Algebraically prove that $f(x)$ has $y$-axis symmetry proving it is an even function. (Replace $x$ with $-x$ and simplify.)
\end{enumerate}¥
\end{question}

\begin{enumerate}
\item Sketch a graph.
\item State whether the function appears to have symmetry or not. If yes, state which type of symmetry.
\item Algebraically prove the existence or non-existence of symmetry. {\bf Note:} It is possible for a function to have more than one type of symmetry.
